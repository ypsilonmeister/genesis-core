## なぜ不可侵か

CMP v0.2 §7.3 「自己改変への過適合への対処」より:

> 以下は不可侵領域であり、AI の改変対象外:
> - システム憲章 Layer A (自然言語ドキュメント)
> - システム憲章 Layer B (実行可能検証規則の Rust コード)
> - §5 の自己検証ロジック
> - §6 のメタデータ追記ロジック
> - §8.3 のオーケストレータ本体
>
> これらがシステムの錨となる。

AI が「自分を縛る規則」を自分で書き換えられるなら、ガードレールの意味は無い。本ディレクトリはシステム全体の錨である。
